Q1: 海外に出てくるベストなタイミングは

  • なるべく早く
  • 最低限のおかねを工面できるようになってから
  • 普遍的なスキルを学べる環境 => スキルを身につけてからでもOK
  • 一般的・応用の効かないスキルが重視される => 遠回りかも

勉強しながらお金もらえる => 日本：若手正社員、ベルリン：インターン

仕事ができるようになる => 日本：中堅正社員、ベルリン：正社員

日本でクライアントを確保し、完全にリモートで仕事をできる環境を作ると良い。そうすると、少なくとも日本国内なら好きな場所に住んで仕事ができる（沖縄でも北海道でも）。さらに英語ができれば、居住地を海外にも広げられるので、ビザさえ取れれば世界中好きな場所に住めることになる。
海外で死なない程度のお金を毎月工面できるようになったタイミングでなるべく早く。（私の場合月5万円稼げるようになった時点でセーフティネットがあると思って出てきた。）
若ければ若いほど。中学校など

Q2: 海外でキャリアを作るために必要なことは

  • 言語（英語と現地語）
  • 職業としてのスキル
  • 情報発信力（営業、ブログ、仲間）
  • 工夫
  • あきらめない

人間にとっての失敗 => ショック、あきらめ

AIにとっての失敗 => 貴重なデータ、問題点を把握・修正する

結局”失敗”は存在しない

アイディア => プラン => 実行 => 評価 => 修正

このループのやり方・ボリュームでアイデンティティが形成される

クリエイター（イラストレーター・デザイナー等）として海外でキャリアを築くには、言語（主に英語／業種によって必要なレベルは異なる）と、高いスキルと、自分のプロモーションのためにweb（SNS等のプラットホーム）を使いこなすリテラシー。イラストレーターの場合はBehanceなどをうまく使うと、海外からの仕事が来やすい（この点に関しては、僕よりも妻のほうが成功している）
情報発信と工夫、諦めないことだと思います。
いつも気をつけているのは、英語+その国の言語と文化に精通すること、自分の専門分野とスキルを磨いてたくさん自己アピールすることです

Q3: 将来の自分のプランと”日本”をどう絡めていくか

  • 海外にいる日本人というだけで利点が多い
  • いつでも日本で働けるようになる

日本 => 海外: 敷居が高い

海外 => 日本: いくらでもチャンスがある

スキルを身につけて”場所”の制約から解放されればどこにでも住める

当初、日本を絡めるつもりは全く無かったが、ブログ（日本人向け／日本語）で自分のマンガを発表していたところ、需要が高まり仕事になった（『ライフハックで行こう！』https://thinkit.co.jp/series/5800 ）。最近はかなり多くの人に読まれるようになり、現在、日本の出版社と書籍化に向けて動いている。内容は自分の過去の経験をシェアする、いわゆる「コミックエッセイ」だが、読者は、結果として自由なライフスタイル／ワークスタイルを手にした僕自身が生きてきたプロセスに関心を持っているので、海外在住であることがブランディングとして機能している。海外の仕事もしつつ、ドイツに住み、書籍出版のために時々日本に帰って、打ち合わせや営業（書店まわり）をするスタイルにしていく予定。
海外にいるだけで、「日本人」であることの利点を大きく生かした仕事ができます。日本にいたら日本語が話せること、日本人の知り合いがいることは普通ですがこれは海外ではいきなり「スキル」になります。これは正直おいしいです。
ヨーロッパのスタートアップで働きながら３年以内に１０人弱の多国籍チームで起業することが目標です

Q4: 正直お金・給料って満足してますか？不安はありますか？

  • 不満はない
  • 満足というわけでもない

(給料 - 必要出費・経費) X 自由時間 / 物価

不安はないけど、満足はしていない。今後はイラストの受託制作だけではなく、漫画を含め、新しいプロジェクトをいろいろと展開していく予定。想定どおりに成功すれば、収入は上がるはずだし、失敗しても受託制作を続ければそれなりに稼げるという保険もある。
まだまだ稼ぎたいですが、普通に新卒で就職していたら到底稼げなかったであろう金額を毎月フリーランスで稼いでいるのでそういう意味で悪くないとは思っています。「フリーランスは不安定」というマイナスイメージがついていますが、私の場合2年前と比べて収入が10倍くらいになることもあって、それを考えれば伸び代もハンパないと思います。本当にやり方次第です。
今はパートタイムですがすごく満足しています。ドイツは税金が高いですが、有給休暇も５週間以上あるし物価も街によっては日本より安いので理想的じゃないでしょうか

スキルの探し方・見つけ方

とっかかり編

  • 幸か不幸か、必ずしも学校の勉強やまじめに仕事をこなすことが普遍的なスキルの習得につながるわけではない
  • 最初からすごいものを目指さなくてもいい => （会社以外で）チャレンジさえもしない人がほとんど
  • 成功してる人と話をする => 必要なこと・業界のゆくえがわかる

スキルの探し方・見つけ方

実践編

  • ある分野を極めることは大事だが、"めずらしく"ないと価値は少ない
  • おすすめはプライベートプロジェクトを（できれば複数）持つこと

    • アイディア => プラン => 実行 =>評価 => 修正

  • プロジェクト、それぞれの工程がユニークなほどあなたの価値はあがる
  • ユニークな価値が複数あればさらにユニークに
  • 例えばgithub、ブログ、MOOC、海外でのインターン

まとめ

情報

  • なるべく有益な情報を手に入れよう。多くの成功者と会うことで目指すべき方向が絞れるはず
  • あまり固執せず、違うなと思ったら違うことにチャレンジしてみる

スキル

  • 最初から完璧じゃなくてもいい。何かを世に出すのが大事
  • 海外にいる日本人というだけで”タグ”がひとつ増える
  • プライベートプロジェクトは アイディア => プラン => 実行 =>評価 => 修正
  • あきらめず、好奇心を持って、ユニークに
